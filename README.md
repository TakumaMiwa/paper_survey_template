# 概要
* 論文執筆用に先行研究論文を管理するためのレポジトリ
* サーベイした論文のうち，論文に引用する可能性があるものを.mdファイルで共有
# ファイルの追加方法
1. papersディレクトリ直下に論文タイトル(またはその略称).mdファイルを作成
2. template.mdファイルをコピペして以下の項目を埋める
    * 論文の位置付け (なぜこの論文を共有するのか e.g. すでに追加されている...という論文の根幹となっており，ベースラインとして...をあげる根拠を述べる上で必要等)
    * 研究背景
    * 提案手法
    * 長所と短所
    * .bibtex
3. pull requestを作成して共同研究者にreviewを求める
